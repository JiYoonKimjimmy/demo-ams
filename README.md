# AMS 

## Academy Management Service Project <small>학원 관리 서비스</small>

- 학원 구성원 및 수업 일정 관리 기능 제공
- 학원 정보 공유 & 추천 서비스 제공
- 학원비 온라인 결제 서비스 제공

---

### Project Features

#### 학원 구성원 정보 관리

- 학생/학부모 정보 관리
- 학원 강사 정보 관리 

#### 수업 정보 관리
 
- 수업 일정 관리
- 수업 학생 출석 관리

#### 학원 수강 정보 관리

- 수강 과목
- 수강 대상
  - 추천 대상 범위
- 수강 기간
- 수강 목표
  - 기본 목표
  - 완강 효과
- 수강 교육과정
- 강사 정보 & 소개
- 수강 & 강사 평가

---

### Project Spec

- Kotlin 2.1.10
- Spring Boot 3.4.3
- Gradle 8.13
- H2 Database

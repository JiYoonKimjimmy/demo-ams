# 🚀 AMS - Academy Management Service

> **학원 관리 서비스 기능 요구 사항서**

---

## 📘 개요

**AMS(Academy Management Service)**는 학원 운영과 수강 관리를 효율적으로 지원하기 위한 RESTful 기반의 백엔드 서비스입니다.  
본 문서는 AMS 시스템 개발을 위한 **핵심 기능 요구 사항(Functional Requirements)** 을 정리한 자료입니다.

---

## 📂 주요 기능

### 1. 학원 정보 관리 (Academy Management)

학원의 기본 정보를 등록하고, 조회 및 수정할 수 있는 기능을 제공합니다.

#### 📄 주요 기능

- 학원 정보 등록 / 조회 / 수정
- 학원 기본 정보 항목:
  - 학원 명칭  
  - 학원 연락처  
  - 학원 주소  
  - 운영 시간  
  - 대상 범위  
  - 교육청 등록번호  

---

### 2. 회원 관리 (Member Management)

학원 서비스 이용자(학생, 학부모, 강사, 관리자 등)의 회원 정보를 통합적으로 관리합니다.

#### 📄 회원 정보 항목

- 이름  
- 휴대폰 번호  
- 생년월일  
- 성별  
- 주소  
- 로그인 정보  
  - 로그인 ID  
  - 비밀번호  

#### 👥 회원 유형

| 회원 구분 | 설명 |
|------------|------|
| **학생(Student)** | 학원 수업을 수강하는 회원 |
| **학부모(Parent)** | 학생의 수강 현황을 조회 및 관리하는 회원 |
| **강사(Teacher)** | 학원 수업을 담당하는 회원 |
| **운영 관리자(Manager)** | 학원의 전반적인 운영을 담당하는 회원 |
| **슈퍼 관리자(Super Admin)** | 전체 학원 시스템을 총괄 관리하는 회원 |

#### 🔐 회원 가입 및 인증

- 서비스 가입은 **휴대폰 인증(6자리 코드 발급 및 검증)**을 통해 진행됩니다.  
- 인증이 완료된 후 회원 정보, 로그인 ID, 비밀번호를 입력하면 가입이 완료됩니다.  
- 로그인은 다음 두 가지 방식을 지원합니다.
  - `로그인ID + 비밀번호`
  - `휴대폰 인증 + 비밀번호`
- 로그인 ID 찾기 및 비밀번호 재설정 기능을 제공합니다.

#### ⚙️ 회원 승인 및 생성 권한

- `강사` 회원의 가입은 소속 학원의 **운영 관리자 승인**이 필요합니다.  
- `운영 관리자`는 `학생`, `학부모`, `강사` 회원을 생성할 수 있습니다.  
- `슈퍼 관리자`는 `운영 관리자` 회원을 생성할 수 있습니다.  
- `학생`과 `학부모`는 **1:N 관계**로 연결됩니다.

#### 🛡️ 역할별 권한

| 구분 | 기능 제한 |
|------|------------|
| **학생** | 수업 정보 관리 기능 제한 |
| **학부모** | 수업 정보 관리 기능 제한 |
| **강사** | 수업 생성 및 삭제 기능 제한 |
| **운영 관리자** | 학생·학부모·강사 정보 수정 및 삭제 가능 |
| **슈퍼 관리자** | 모든 기능 이용 가능 |

### 3. 수업 관리 (Class Management)

수업 정보와 일정, 출석, 평가 등 학원 수업 전반을 관리합니다.

#### 📘 수업 정보 관리

- 수강 과목  
- 수강 대상  
- 수강 기간  
- 수강 목표  
- 교육 과정  
- 강사 소개  
- 수강료  

#### 📅 수업 일정 관리

- 수업 명칭  
- 수업 과목  
- 수업 강사  
- 수업 요일  
- 수업 기간  
- 수업 대상  
- 수업 정원  
- 강의실 정보  
  - 강의실 명칭  
  - 강의실 위치  

#### 🧾 출석 관리

- 학생은 수업 출석 시 **출석 확인 요청**을 보냅니다.  
- 강사는 요청 목록을 확인하고 **출석 확인**을 완료합니다.  
- 출석이 완료되면 학부모에게 **알림이 발송**됩니다. *(알림 인터페이스만 구현 예정)*  
- 출석이 확인되지 않은 경우에도 **자동 알림 발송** 기능이 동작합니다. *(인터페이스만 구현 예정)*  
  - 수업 시작 시각 기준으로 출석 미확인 학생 목록을 자동으로 조회하여 알림을 발송합니다.

#### 🌟 수업 평가 관리

- `학생`은 수강한 수업 및 강사에 대한 평가 게시글을 작성할 수 있습니다.  
- `학부모`는 자녀가 수강한 수업 및 강사에 대한 평가 게시글을 작성할 수 있습니다.  
- `강사`는 평가 게시글을 수정하거나 삭제할 수 없습니다.

---

### 4. 운영 관리 (Operation Management)

학원의 수납, 영수증, 통계 등 운영과 관련된 기능을 제공합니다.

#### 📊 주요 기능

- 수납 관리  
- 영수증 발급  
- 통계 제공  
  - 기간별 통계  
  - 수업별 통계  
  - 강사별 통계

## 🔧 향후 개선 사항 (Future Enhancements)

- **OAuth 2.0** 기반의 소셜 로그인 기능 추가  
- **알림 발송 기능 고도화**  
  - `카카오 알림톡`, `SMS`, `PUSH` 지원 예정  
- **위치 기반 자동 출석 기능**  
  - 학생의 위치가 학원 좌표 반경 50m 이내일 경우 자동 출석 처리  
- **비대면 결제 기능 추가**  
  - 온라인 수강료 결제 시스템 도입
